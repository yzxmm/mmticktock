# 资源需求表

为了获得最佳的手绘效果，请提供以下 PNG 图片文件，并将它们放在程序的 `assets` 文件夹中。

## 1. 基础图片
所有图片背景必须是透明的。

| 文件名 | 描述 | 建议尺寸 | 备注 |
| :--- | :--- | :--- | :--- |
| `bg.png` | 整体背景图 | 800x400 px | 这是一个容器背景，数字将显示在其中央。可以是手绘的方框、云朵等。尺寸越大越清晰。 |
| `colon.png` | 冒号分隔符 (:) | 200x300 px | 手绘风格的冒号。建议与数字图片尺寸一致，内容居中。 |

## 2. 数字图片
需要 0 到 9 的手绘数字。确保它们的风格、大小一致，以便排列时整齐。建议使用较大的尺寸以保证缩放后的清晰度。

| 文件名 | 描述 | 建议尺寸 |
| :--- | :--- | :--- |
| `0.png` | 数字 0 | 200x300 px |
| `1.png` | 数字 1 | 200x300 px |
| `2.png` | 数字 2 | 200x300 px |
| `3.png` | 数字 3 | 200x300 px |
| `4.png` | 数字 4 | 200x300 px |
| `5.png` | 数字 5 | 200x300 px |
| `6.png` | 数字 6 | 200x300 px |
| `7.png` | 数字 7 | 200x300 px |
| `8.png` | 数字 8 | 200x300 px |
| `9.png` | 数字 9 | 200x300 px |

## 3. 目录结构示例
```
ProjectRoot/
  |-- tiktak.exe
  |-- assets/
      |-- bg.png
      |-- colon.png
      |-- 0.png
      |-- 1.png
      ...
      |-- 9.png
```

## 4. 注意事项
- 图片边缘请处理干净，保持半透明阴影效果会更好。
- 如果不提供背景图 (`bg.png`)，程序将只显示数字。
- 程序会自动居中排列这些数字。
